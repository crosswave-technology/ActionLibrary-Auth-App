## Summary

Resolves GH-001: all external GitHub Action references in this repo
converted from mutable tag refs (`@vX`) to immutable SHA digest refs
(`@<sha> # vX`).

Also adds .actionlint.yaml at repo root (suppresses two narrow false
positives in our standard workflow templates - PR number and commit SHAs
used directly in 
un: blocks).

### How to update a pinned action

1. Identify the new tag (e.g. `v4.3.0`)
2. Run: `gh api repos/<owner>/<repo>/git/ref/tags/<tag>`
3. Follow annotated tag if needed (`type == tag`)
4. Replace the SHA in the workflow file and update the `# vX` comment
